# Portfolio Site

## 概要
Nuxt3とVue.jsを使用したモダンなポートフォリオサイトです。シンプルで洗練されたデザインと、スムーズなアニメーションを特徴としています。

## 技術スタック
- **フレームワーク**: Nuxt3
- **言語**: TypeScript
- **スタイリング**: TailwindCSS
- **アニメーション**: カスタムトランジション
<!-- - **デプロイ**: Cloudflare Pages -->

## 主な機能
- レスポンシブデザイン
- ページトランジションアニメーション
- スクロールアニメーション
- 作品一覧表示
- コンタクトフォーム
